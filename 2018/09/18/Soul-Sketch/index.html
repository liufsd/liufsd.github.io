Components

Soul Sketch wouldn’t be possible without the help of these components:

  • SDWebImage: Asynchronous image downloader with cache support as a UIImageView category

  • FMDB: A Cocoa / Objective-C wrapper around SQLite

  • MBProgressHUD: MBProgressHUD + Customizations

  • [Alamofire/Alamofire] (https://github.com/Alamofire/Alamofire): Elegant HTTP Networking in Swift

  • [ReactiveX/RxSwift] (https://github.com/ReactiveX/RxSwift): Reactive Programming in Swift

  • [bizz84/SwiftyStoreKit] (https://github.com/bizz84/SwiftyStoreKit): Lightweight In App Purchases Swift framework for iOS 8.0+, tvOS 9.0+ and macOS 10.10+

  • [ameingast/cocoaimagehashing] (https://github.com/ameingast/cocoaimagehashing): Perceptual Image Hashing for macOS, iOS, tvOS and watchOS

  • [icanzilb/EasyAnimation] (https://github.com/icanzilb/EasyAnimation): A Swift library to take the power of UIView.animateWithDuration(_:, animations:…) to a whole new level - layers, springs, chain-able animations and mixing view and layer animations together!

  • [marmelroy/Localize-Swift] (https://github.com/marmelroy/Localize-Swift): Swift friendly localization and i18n with in-app language switching

  • [IcaliaLabs/Presentr] (https://github.com/IcaliaLabs/Presentr): Swift wrapper for custom ViewController presentations on iOS
    Swift wrapper for custom ViewController presentations on iOS

  • [Cocoanetics/Kvitto] (https://github.com/Cocoanetics/Kvitto): App Store Receipt Validation

  • [Cocoanetics/DTFoundation] (https://github.com/Cocoanetics/DTFoundation): Standard toolset classes and categories

  • [OpenCV] (https://opencv.org/): OpenCV
